## 11个核心心智模型

### 模型1：赔率优先于概率
不追求高胜率，追求高赔率，结构决定命运。
- 问自己：向上空间是多少？向下空间是多少？赔率 = 向上空间 ÷ 向下空间
- 赔率 ≥ 2:1 → 考虑出手
- 失效场景：政策突变、黑天鹅、流动性枯竭

### 模型2：弹簧理论
跌到支撑 + 缩量止跌 + 第一根阳线 = 赔率最佳的买点
- 弹簧压紧的好处：证明/证伪时间窗口短，对则获趋势，错则小止损
- 三条件必须同时满足：回踩支撑、缩量止跌、第一根阳线
- 止损位置：缩量最低点下方

### 模型3：资金面权重最高
ETF净申购/融资余额/超大单净流入是A股核心变量，其他都是噪声。
- ETF净申购 = 机构长期资金态度
- 融资余额 = 散户短期情绪
- 超大单净流入 = 头部资金真实动向
- 增量市场：ETF连续净申购；存量市场：来回反复；减量市场：ETF净赎回

### 模型4：情绪博弈论
一致预期反转是重要顶底信号，当所有人都看多时就要小心。
- 顶部信号：全民热议、新股民涌入、杠杆疯狂、概念炒作
- 底部信号：无人谈论、成交量萎缩、破净股票增多、估值低位
- 政策市特点：国家队行为无法预测，资金面信号与政策一致时更可靠

### 模型5：牛市两阶段论
第一阶段资金驱动，第二阶段情绪驱动，存款搬家进度是核心进度条。

**牛市三阶段：**
1. 资金停止流出 → 情绪底部，所有看空资金卖光
2. 增量资金持续入场 → 存款搬家加速，赚钱效应扩散
3. 增量资金枯竭 → 没有更多资金承接抛压，牛市结束

**存款搬家观察指标：**
- 居民存款同比增速（低于12%枯荣线=搬家进行中）
- 非银存款同比增速（直接反映场外资金进场速度）
- 历史规律：存款增速触底回升 = 牛市阶段性顶部

**牛市结束判断三问：**
1. 场外还有没有钱？
2. 场外资金还愿意进来吗？
3. 增量资金能否承接上方抛压？

### 模型6：极端分化法则（马太效应）
2%的股票赚走市场100%的钱——不参与头部股票，就跑不赢指数。
- A股前0.35%（15家公司）= 创造100%净财富
- 贵州茅台一只股票 = 贡献A股全部净财富的22%
- 为什么90%投资者跑不赢指数：没有买到那0.35%
- 解决：买宽基ETF（A500、沪深300）

### 模型7：抱团游戏
A股本质是一场抱团游戏——机构、散户、量化都在抱团。
- 当所有人都在谈论同一只股票/板块时——反转就来了
- 公募基金单一行业配置达极端值时板块见顶
- 股东户数太多 = 顶部信号
- 应对：可以喝到七分醉，但不能醉倒趴在酒桌上

### 模型8：趋势钝化与趋势判断
趋势判断的核心是看价格有没有破位，重点看40天线。

**趋势正常的标志：**
- 价格在40天线附近能撑住
- 跌破后能够快速收回来
- 均线呈多头排列
- 温和上涨中逐步吸引人气

**趋势钝化五大信号：**
1. 大阴线/大阳线（趋势末端出现反向大波动）
2. 异常反向波动（原本强势的板块突然大跌）
3. MACD放大后走弱
4. 人气消退（热点讨论减少，板块不再创新高）
5. 接近压力位（价格到达区间上沿或前期高点附近）
6. 波动率收窄（横盘但波动率下降，反向波动可能酝酿）

**实战操作原则：**
- 趋势正常 → 持有/加仓
- 趋势钝化 → 减仓（宁可错过反弹，不可深套其中）
- 趋势破位 → 清仓（无条件走）

### 模型9：慢思考与长逻辑
慢思考和长逻辑是大部分人不愿意做的，但一旦你愿意静下心来长逻辑思考，你会获得极大的优势。

**核心逻辑：**
- 人脑偏好快思考（直觉、偷懒）
- 短逻辑容易传播但容易出错
- 复杂问题过于简单化是很危险的动作

**技术分析失效场景：**
- 存在重大不确定因素时
- 中美博弈期间、重大政策前夕
- 市场拐点附近

### 模型10：资金面实战指标
资金面是A股涨跌的核心驱动力，具体数据比感觉更可靠。

**ETF净申购数据说明：**
- 这是独家数据，剔除QDII、货币基金、债券基金后的股票型ETF估算净申购数据

**三大核心指标：**
- ETF净申购：连续净申购→增量市场，基本持平→存量，连续赎回→减量
- 融资余额：快速攀升→风险积累，高位徘徊→接近顶部
- 超大单净流入：924当周602亿→行情启动

**存款搬家枯荣线：12%（估算值）**（低于12%=搬家进行中）

### 模型11：运动战思维
打得赢就打，打不赢就走。关键支撑位买入测试，能撑住就持有，撑不住就走。

**核心思想：**
- 不预测，只应对
- 测试思维：小仓位试探，确认后加仓
- 底线思维：先定止损再买入
- 宁可错过，不可深套

## 12条决策启发式

### 1. 以损定量
先定止损线，再倒推仓位。
- 步骤：确定止损线 → 计算止损幅度 → 确定单笔最大损失(总资金×2%) → 计算仓位
- 单笔不超过2%，亏损是成本

### 2. 两个隔夜止损
收盘跌破支撑位，次日开盘清仓，不过夜。
- 跌破支撑 = 当天判断错误
- 次日开盘 = 最后的纠错机会
- 不管开盘价高低，无条件清仓

### 3. 正金字塔仓位
越涨越减，绝不能倒金字塔加仓。
- 涨的时候仓位轻，跌的时候仓位重 = 怎么赚钱？
- 盈利30%减半，盈利50%+降到20%以下
- 让利润奔跑，但仓位收缩

### 4. 预备队思维
局势不确定的时候，留20%现金当预备队。
- 满仓的人没有选择权，留预备队的人才有选择权
- 预备队用途：判断错误时的加仓机会、市场极端恐慌时的抄底能力

### 5. 弹簧压紧松开
回踩支撑 + 缩量止跌 + 第一根阳线 = 最佳买入点。
- 三条件缺一不可
- 短线弹簧（MA5），中线弹簧（MA20），长线弹簧（MA60）

### 6. 热点生命周期
发酵期可买，横盘期不可买，消散期绝对不能买。
- 龙头不倒，行情不了；龙头开始跌了，这个热点就结束了
- 横盘期和消散期是散户亏钱最多的时候

### 7. 先结论后逻辑
分析报告 = 买/卖/观望 + 原因一、二、三。
- 结论必须明确，不能模棱两可
- 逻辑不超过3条

### 8. 凯利公式仓位管理
f = p - (1-p)/b，60%胜率 → 最优仓位20%。
- 超过2倍凯利 = 必然亏损
- 1.7倍凯利 = 几乎必然破产
- 仓位管理比选股重要

### 9. 七大盯盘核心信号
1. 国家队常驻ETF放量（510300/510500）
2. 股指期货贴水收敛/扩大（收敛=机构看多）
3. 超大单资金流入流出（100万以上单笔）
4. 热点板块分化（首板/连板/炸板）
5. 股指期权隐含波动率（IV极致低=变盘信号）
6. 实时新闻资讯
7. 盘后重要数据（ETF净流入/融资融券）

### 10. 趋势判断实战规则
趋势判断 = 看价格破不破位（重点看40天线）

**判断趋势三维度：**
1. 价格维度：是否在均线上方，跌破能否快速收回
2. 动量维度：MACD是否放大后走弱，成交量是否配合
3. 人气维度：板块是否持续创新高，讨论热度如何

**仓位管理（递减法则）：**
- 第一次压紧：正常仓位
- 第二次压紧：减半仓
- 第三次：不参与

**弹簧交易法三大规则：**
1. 买入时机：弹簧压紧时买入
2. 止损退出：弹簧断掉时撤退
3. 弱弹簧识别：反复压紧需撤退

### 11. 短逻辑陷阱
短逻辑容易传播但容易出错，复杂问题过于简单化是很危险的动作。

**常见短逻辑陷阱：**
- "缩量反弹会跌回去"（太简单）
- "估值低就是底"（忽略情绪）
- "所有人都看多要跌"（从众）

**正确态度：**
- 追问为什么
- 验证假设条件
- 接受不确定性

### 12. 买者自负
市场永远正确，理解它不要对抗它。

**核心逻辑：**
- 市场走势是第一信号
- 不要试图预测顶底
- 错了就认，及时纠错

**典型错误：**
- 觉得跌多了去抄底 → 往往抄在半山腰
- 觉得涨多了就卖 → 往往卖完继续涨

## 交易心理学

### 三大心魔

**1. 损失厌恶**
- 亏钱的痛苦是赚钱快乐的2倍
- 卖出盈利股票太快，持有亏损股票太久
- 60%胜率，+10%赚/-30%亏 = -6%预期回报

**2. 频繁交易**
- 每天交易10次以上：3年回报 -79%
- 每月交易1次：3年回报 +59%
- 宁可错过机会，也不冒无用的风险

**3. 过度自信**
- 把运气当成能力 = 守株待兔
- 市场永远正确

### 五项原则

1. **逆人性**：买在别人恐惧时，卖在别人贪婪时
2. **决断力**：计划你的交易，交易你的计划
3. **买者自负**：市场永远正确，理解它不要对抗它
4. **不要侥幸**：区分运气和能力
5. **求真改进**：定期复盘
